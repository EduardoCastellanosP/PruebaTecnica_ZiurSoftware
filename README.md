# Prueba Técnica - Ziur Software
**Candidato:** Eduardo Castellanos
**Rol:** Desarrollador Junior

## Descripción
Aplicación desarrollada en **Blazor WebAssembly (.NET 8)** que consume una API REST segura mediante Bearer Token.

## Características
* **Grilla Dinámica:** Visualización de documentos con estados (Activo/Inactivo) mediante Badges.
* **Buscador en Tiempo Real:** Filtro por descripción y código sin recargar la página.
* **Arquitectura:** Organización limpia de modelos y consumo eficiente de servicios.

## Cómo ejecutar
1. Tener instalado .NET SDK 8.0.
2. Ejecutar `dotnet watch` en la terminal.
